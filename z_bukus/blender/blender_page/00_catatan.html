//======================================================================================================================== //---------------> Modeling (1)

object mode:
ctrl+j : menggabungkan mesh menjadi satu
ctrl+p : membuat parent (select kedua object)
shift+a : menambahkan tulang

//> edit mode:
tab : ganti mode (object/editmode)
shift+a : menambahkan object primitive
ctrl+r : memotong loop (scroll untuk menambah potongan)
i : inset faces (extrude dalam mendatar)
x : menghapus
alt+n : memunculkan menu untuk menormalkan dengan membalik mesh(cth.flip, recalculate outside/inside, smooth/reset vector)
z : untuk mengganti shading
m : merge menggabungkan yang di select
e : extrude (melebarkan mesh)
k : cut (memotong mesh)
ctrl+b : bevel (memperhalus sisi)
v : memotong vertec
p : sparate (memisahkan mesh menjadi mesh baru)

//======================================================================================================================== //---------------> uv editing (2)

u : unwrap
alt+r : me refresh gambar di un editing
fitur:
mark seam : membantu memotong untuk wrap

//======================================================================================================================== //---------------> rigging (3)

//> object mode : >1
shif+a > armature > single bone : menambahkan satu tulang utama
object data properties > viewport display > in front : agar tulang di terlihat di depan

//> edit mode : >2
*sesuaikan tulang gunakan (shit+d) untuk duplikat tulang / (e) untuk extrude tulang
*gunakan (ctrl+p) untuk menghubungkan tulang
*jika inggin membagi tulang (click right) kemudian pilih (subdivide) tinggal atur jumlah bagianye
*jika inggin menduplikat tulang ke sisi satunye rename dengan format (nama.l/nama.r) kemudian (click right) pilih (symmetrize)

//> object mode : >3
*jika sudah seleksi (mesh karakter) kemudian seleksi (armature) kemudian (ctrl+p) pilih (with empty group / bisa pilihan yang lain sesuai kebutuhan)
